# 关于本校四六级(CET)相关的介绍

:::info

##  本介绍致力于给对四六级有疑问的同学解疑

:::

## 四六级相关介绍

### 一、考试科目

考试科目：

```
    英语四级考试（CET4）  09:00-11:20
```

```
    英语六级考试（CET6）  15:00-17:25         
```

### 二、报名对象

在校各年级本科学生和硕士研究生

### 三、报名费用

根据皖价费函〔2012〕124号，CET4报名费`28元/人`，CET6报名费`30元/人`。

### 四、报名时间和缴费办法

1. 网上统一报名时间为：`以实际教务处通知为准`

2. 网上报名系统网址为:http://cet-bm.neea.edu.cn

:::tip

学生登录报名网站，浏览通知与公告、报名流程以及常见问题解答等，按照网站提示自助完成报名操作。报名完成后，根据系统提示在线完成缴费。学生也可以下载安装“CET”手机APP，进行报名及缴费。

:::

3. 本次考试报名系统新增考位候补功能。考点考位报满后，考生可选择“候补”报名，报名系统将按照“候补”报名时间顺序，依次填补未按时缴费考生空出的考位。

### 五、准考证打印

报名且缴费成功的学生，自行登陆报名网站，下载打印本人CET准考证。

### 六、成绩报告单

1. 成绩发布10个工作日后，学生可登录[中国教育考试网](http://www.neea.edu.cn)查看并下载电子版成绩报告单。电子版成绩报告单与纸质版同等有效。

2. 纸质成绩报告单依申请发放。学生可在报名期间或成绩发布后规定时间内登录[CET报名网站](http://cet-bm.neea.edu.cn)，自主选择是否需要纸质成绩报告单。纸质成绩报告单在考后规定时间段由考点学校统一发放。

### 七、注意事项

:::caution 注意

1. 考生报名和缴费应在规定时间内完成，逾期不予受理。

:::
:::info 关注

2. 用户注册信息中姓名、身份证号与本人学籍信息必须完全一致，否则无法报考。若有以下情况，学生须联系所在学院：

(1) 学生符合报考条件，但未查询到报考资格；

(2) 学生的学校、院系、班级、所属校区等基本信息以及照片等与本人实际情况不符。

以上情况由学院统一收集学生电子照片（jpg格式，大小不能超过200kb，以“身份证号码”命名）和学籍信息（见附件模板），汇总后于11月2日（星期三）前发至教务处进行处理；CET报名不接受学生个人提交的任何材料。

:::
:::note

3. CET报考资格由报名系统自动审核。CET4成绩达到425分及以上的在校学生，方可报考CET6。

:::
:::note

4. 我校考点最大考位容量为6950人（南区5300人，北区1650人），考位报满后将不再接受报名。已通过CET的学生不可重复报考。

:::
:::danger 警告

5. 英语四六级考试无故缺考、违纪的学生本次停考一次。本次CET无故缺考、违纪的学生将在下一次考试时停考一次。

:::
:::tip

俩种情况禁止入场

1. 入场开始15分钟（即上午09:00，下午15:00）后禁止入场。

2. 考试证件如有遗失，请提前办理或开具相关证明，证件不齐者不得进入考场

:::
